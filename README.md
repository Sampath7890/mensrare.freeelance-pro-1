# Mensrare Ecommerce Store

Production-oriented Vite storefront for premium menswear.

## Development

```bash
npm install
npm run dev
```

## Production

```bash
npm run build
npm run preview
```

Optional Supabase environment variables:

```text
VITE_SUPABASE_URL=
VITE_SUPABASE_ANON_KEY=
```

See [PERFORMANCE_AUDIT.md](./PERFORMANCE_AUDIT.md) for the bottleneck analysis, architecture, bundle measurements, deleted files, and estimated Lighthouse impact.
